# ScenifyXR

**"Say it. See it. Train in it."**

AI-powered, **procedural XR scene & asset generator** from natural language — **no prefab library required**.

- **Input**: `"Create a hospital server room with 2 Cisco 9300 switches and a red-blinking UPS"`
- **Output**: Fully interactive Quest 3 scene in **<60 seconds**
- Built with **Unity 2023.2 LTS + XR Interaction Toolkit + GPT-4o**

---

## Features
- Procedural prefabs (primitives + labels + ports + faults)
- On-the-fly XR interaction injection
- Deployable to **Meta Quest 3**
- 100% open-source, no external 3D assets

---

## Quick Start

1. Clone: `git clone https://github.com/yourname/ScenifyXR.git`
2. Open in **Unity 6.2 LTS**
3. Import **XR Interaction Toolkit → Starter Assets**
4. Play → Use `ScenifyXR` menu to generate scenes
